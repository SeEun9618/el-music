# 정세은

## 프로젝트 구동 방법

```bash
$ npm i
$ npm run build
$ npm run start
```
## prettier 적용
```bash
$ npx prettier --write .
```

## 프로젝트 설명

### 프로젝트 개요
- 음원사이트 이엘뮤직 UI 및 기능 구현

### 주요 기능
- TOP100 차트
- 음원 상세 페이지
- 음원 검색
- 음원 정렬

### 기술 스택
- Node (v16.17.x)
- Next.js (v13.3.0)

### 주요 라이브러리
* axios
* styled-components
